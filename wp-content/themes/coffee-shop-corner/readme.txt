== Description ==

Coffee Shop Corner is a beautiful and elegant theme. With this, anyone can kick start their online appearance. Any beginner can use this theme and can showcase their products and services. Coffee Shop Corner is suitable for any kind of beverage shop like a cafeteria, coffee bar, steam room, tea shop bistro, brasserie, wine bar, café bar, cybercafé, anti café, milk bar, etc. And also can be used by Coffee Roasters, Cafés, Coffee Shops, Tea shops, Bakeries, and Restaurants. The theme has amazing features such as Clean, secure code, Retina ready, user-friendly, responsive, and many other professional support features. The theme offers a wide range of features to help the users create a stunning website fast and easily and with no coding skills. The theme is bright and interactive, with a clean design. Visitors can browse menus and services quickly. This theme is perfect for coffee shops, beverage websites, personal baristas, coffee shops, and other coffee-related businesses. The Multipurpose theme is flexible enough to meet your needs. It has various powerful features allow you to create a professional coffee portfolio. You can also display your business and menus and include all the other services your coffee shop offers. You can use it for both presentational and custom shortcodes. They are perfect for displaying the essence of your business.

For Homepage Setup:
====================

1. Go to Dashboard >> Pages >> Add New, Create a new page and name it as "Home" then select the template "Custom Home Page" and publish it.

2. Go to Dashboard >> Settings >> Reading, Select the option of Static Page, now select the page you created to be the homepage.

For Header Section:
====================

1. Go to Dashboard >> Appearance >> Customizer >> Theme Settings >> Header, add contact button text & its url and publish it.

For Slider:
============

1. Go to Dashboard >> Pages >> Add New, Create a new Page and name it, add the content, featured image and publish the page. Repeat this step for other sliders.

2. Go to Dashboard >> Appearance >> Customizer >> Theme Settings >> Slider Settings, Click the checkbox of Show/Hide Slider, select the pages of the slider that you have created and publish it.

For Service Section:
====================

1. Go to Dashboard >> Posts >> Category, add new category for services, name it and publish it.

2. Go to Dashboard >> Posts >> Add New, Create a new Post and name it, assign the service category and publish the post. Repeat the step for other service posts.

2. Go to Customizer >> Theme Settings >> Service Section, add section small title, add section title select the service category that you have created and publish it.

For Footer Section:
====================

1. Go to Dashboard >> Appearance >> Widgets, here you can add widgets to the footer widgets areas, which can be seen in footer.

For Copyright Text:
====================

1. Go to Customizer >> Theme Settings >> Footer Text, add footer text here and it also have an option to show/hide back to top.
